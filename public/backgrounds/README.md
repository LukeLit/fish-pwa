# AI-Generated Background Images

This directory stores background images generated via Vercel AI Gateway (Google Imagen or Flux).

## How It Works

1. Use DevTools to generate underwater scenes, ocean depths, or tropical reefs
2. Images are saved here automatically when you click "Save BG"
3. Backgrounds can be used as game scenes or for composite previews

## Usage

Generated backgrounds are accessible at `/backgrounds/filename.png` and can be used in your game scenes.

## File Naming

Format: `bg_{timestamp}.png` or `composite_{timestamp}.png`

## Models

- Google Imagen 4.0 Fast (recommended)
- Google Imagen 4.0 (higher quality)
- Black Forest Labs Flux 2 Pro
